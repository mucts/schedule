# schedule

>排队系统，预约号源池

### 目录结构
```
app -- 框架程序
    controllers -- 控制器
    lib -- 库
    middleware -- 控制器
    models -- 数据库模型 
    plugins -- 插件
    policies -- 策略
    routes -- 路由
    services -- 服务
    utils -- 工具
bootstrap -- 项目引导程序
    index.js -- 引导入口
config -- 配置文件
    config.js -- 公共配置
    constant.js -- 常量定义
    database.js -- 数据库配置
    log.js -- 日志配置
    route.js -- 路由入口
    index.js -- 配置入口
index.js -- 入口
```
